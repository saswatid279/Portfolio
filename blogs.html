  1. React Context API

    Context API provides a way to share data among multiple components without explicitly passing data at every level i.e., via each component.In React, we break the application into components. These components are isolated pieces of code which are structured like a tree. All the components are connected to a root node. The data is passed among these components in one direction - from top level components to lower level components.

  2. Understanding Closures in Javascript

    A Closure is a function bundled together with it's lexical environment. In other words, it allows an inner function to access the functions and variables of it's parent. Irrespective of the scope where the function is executed, it remembers its outer lexical environment i.e., variables and functions of its parent.
